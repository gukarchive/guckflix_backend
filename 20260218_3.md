## 오늘 사용자 질문 핵심과 답변 요약
- Qdrant가 String을 못 받는가?
  - 답: Qdrant는 int/UUID 지원. 문제는 Spring AI 어댑터의 UUID 가정 경로.
- int id 그대로 유지 가능한가?
  - 답: 네이티브로는 가능하나 Spring AI 표준 검색 경로와 충돌 가능.
- UUID 자동 생성되나?
  - 답: `Document.id`를 지정하지 않으면 Spring AI가 UUID 자동 생성.
- initialize-schema가 재기동 시 데이터를 지우나?
  - 답: 아니며, 컬렉션이 없을 때만 생성.
- 벡터DB 트랜잭션 가능한가?
  - 답: RDB급 ACID 트랜잭션은 일반적으로 제한적. `pgvector`는 PostgreSQL 트랜잭션 활용 가능.
- 느린 구간은 어디인가?
  - 답: `ask` LLM 호출이 최대 병목, 다음 `analyze`, 마지막 `similaritySearch`.
